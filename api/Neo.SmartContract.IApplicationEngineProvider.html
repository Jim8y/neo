Create(TriggerType, IVerifiable, DataCache, Block, ProtocolSettings, long, IDiagnostic)

Creates a new instance of the ApplicationEngine class or its subclass. This method will be called by Create(TriggerType, IVerifiable, DataCache, Block, ProtocolSettings, long, IDiagnostic).

Declaration
ApplicationEngine Create(TriggerType trigger, IVerifiable container, DataCache snapshot, Block persistingBlock, ProtocolSettings settings, long gas, IDiagnostic diagnostic)
Parameters
Type Name Description
TriggerType trigger

The trigger of the execution.
IVerifiable container

The container of the script.
DataCache snapshot

The snapshot used by the engine during execution.
Block persistingBlock

The block being persisted. It should be null if the trigger is Verification.
ProtocolSettings settings

The ProtocolSettings used by the engine.
long gas

The maximum gas used in this execution. The execution will fail when the gas is exhausted.
IDiagnostic diagnostic

The diagnostic to be used by the ApplicationEngine.
