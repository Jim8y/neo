Class ContractPermission

Represents a permission of a contract. It describes which contracts may be invoked and which methods are called. If a contract invokes a contract or method that is not declared in the manifest at runtime, the invocation will fail.

Contract

Indicates which contract to be invoked. It can be a hash of a contract, a public key of a group, or a wildcard *. If it specifies a hash of a contract, then the contract will be invoked; If it specifies a public key of a group, then any contract in this group may be invoked; If it specifies a wildcard *, then any contract may be invoked.
